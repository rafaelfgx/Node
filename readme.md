# Node

API using Node, Express, MongoDB, ESLint, Prettier, Folder-by-Feature Structure.

## TECHNOLOGIES

* [Node](https://nodejs.org)
* [Express](https://expressjs.com)
* [MongoDB](https://www.mongodb.com)
* [ESLint](https://eslint.org)
* [Prettier](https://prettier.io)

## RUN

<details>
<summary>Visual Studio Code</summary>

#### Prerequisites

* [Docker](https://www.docker.com/get-started)
* [Node](https://nodejs.org)
* [Visual Studio Code](https://code.visualstudio.com)

#### Steps

1. Open the **directory** in **Visual Studio Code**.
2. Run the command **docker compose up -d** in the **Terminal**.
3. Run the command **npm i** in the **Terminal**.
4. Run the command **npm run start** in the **Terminal**.
5. Press **F5** (only for debugging).
6. Open **<http://localhost:8000>** in the **Web Browser**.

</details>

<details>
<summary>Docker</summary>

#### Prerequisites
